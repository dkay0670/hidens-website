HIDNet Web

This is a collection of web services, by yours truly ;).

Currently, the web server hosts this website, The MC 1.19.x server's dynmap, a private Vaultwarden instance, and 2 sites for Kommunet (ie.best and norton.com).

HIDNet Chat Relay

This is a Chat relay bot for all HIDNet groups/servers/guilds/whatever. Uses Matterbridge as a base.

Currently, the only services linked together are Discord and IRC, although Telegram and Matrix are also planned.

HBot

If you haven't read the projects page for this yet, HBot is a general purpose Discord bot. It's a fork of NetRunner, which is a fork of WinBot.

If you want to invite it to my server, contact me.
